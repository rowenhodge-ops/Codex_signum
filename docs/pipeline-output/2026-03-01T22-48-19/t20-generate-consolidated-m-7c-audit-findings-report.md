## Core Morphemes

Codex Signum implements six fundamental morphemes that encode all system state:

- **Seed** — Atomic compute units (LLM instances, functions, individual operations)
- **Bloom** — Scoped compositions with defined boundaries (patterns, workflows)  
- **Line** — Directed connections encoding flow and transformation
- **Resonator** — Coupling points that encode relationships and dependencies
- **Grid** — Structural lattices providing spatial organization and clustering
- **Helix** — Temporal spirals encoding evolution and learning trajectories

These morphemes replace traditional "agent/model/pattern" terminology to avoid anthropomorphic framing. AI models are substrate (Seeds), not autonomous participants.
