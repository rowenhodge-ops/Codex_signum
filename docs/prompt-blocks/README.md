# Prompt Blocks

Reusable sections for inclusion in Claude Code session prompts.

| Block | When to Include | File |
|---|---|---|
| Pipeline Supervision | Any session that runs the Architect or LLM-backed pipelines | `pipeline-supervision.md` |

## Usage

Copy the relevant block into the session prompt. These are not automatically included — the prompt author (Ro or the session architect Claude) selects which blocks apply to each session.
