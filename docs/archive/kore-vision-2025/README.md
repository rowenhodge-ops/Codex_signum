## Strategic Context

These documents describe an ambitious **consumer-facing SaaS platform vision** for Kore with:
- Public plugin marketplace (MCP ecosystem)
- Astral Script visual language for end users
- Multi-currency economic layer
- Privacy-preserving federated learning
- Consumer product features (mobile app, chat interface)

**Decision**: Rowen Hodge (Principal) decided **NOT to build consumer SaaS platform**. Instead, focus on:
- **Codex Signum**: AI-native consulting practice productivity platform
- **Internal use only**: 6 autonomous agents, Neo4j graph intelligence, LiteLLM routing
- **Consulting-relevant patterns**: Hierarchical router, hybrid RAG, semantic knowledge graph
- **Revenue-first**: $500k Year 1 target via consulting services, NOT software sales

**Quote** (from conversation):
> "I wont be building a SaaS platform. i just want to super charge codex signum. I will move out of obsidian asap. $65pm for Neo4J is nothing given the revenue i should generate. I want a high grade solution. It not only helps my productivity and quality, it helps me be a better trusted advisor."
