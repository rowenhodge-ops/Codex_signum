# Contributing

Thank you for your interest in Codex Signum.

This project is currently maintained by a sole developer and is **not accepting external contributions** including pull requests, feature requests, or code changes.

You are welcome to:
- **Read** the source code and documentation
- **Fork** the repository for your own experimentation
- **Reference** the work in your own research (with attribution per the Apache 2.0 license)
- **Report security vulnerabilities** via the process described in SECURITY.md

If you have questions about the research or framework, you can open a Discussion (if enabled) or reach out via the contact information in the README.
